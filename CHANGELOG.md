# Changelog

OS32 の変更履歴。

## v0.1 Alpha — 最初の公開リリース

### カーネル (73KB)

- i386 プロテクトモード + ページング + ガードページ
- IDT/PIC 完全制御 (IRQ再マッピング)
- kmalloc カーネルメモリアロケータ
- KernelAPI v26 (118エントリ)
- プログラム終了時のリソース自動回収 (FD/リダイレクト/パイプ/共有メモリ)

### ファイルシステム

- VFS 仮想ファイルシステム基盤
- ext2 読み書き対応
- FAT12 読み込み対応
- パイプバッファ (`|`)、リダイレクト (`>`, `>>`, `<`, `2>`)

### デバイスドライバ

- KBD (uPD8251A), IDE (HDD), FDC (fd0)
- Serial (RS-232C), FM (YM2203), RTC (uPD4990A), KCG (漢字ROM)

### シェル

- 外部プログラム方式の高機能シェル
- Tab補完、コマンド履歴、ワイルドカード、環境変数
- スクリプトエンジン (`if`/`else`/`for`/`while`/`goto`)
- `/etc/profile` による起動時自動設定

### コマンド (v0.1 新規追加)

- `sleep` — 指定秒数ウェイト
- `touch` — 空ファイル作成
- `hexdump` — 16進ダンプ表示
- `find` — ファイル名検索 (再帰走査)
- `sort` — 行ソート (-r/-n オプション)
- `du` — ディスク使用量表示
- `cal` — カレンダー表示 (RTC連携)
- `diff` — 簡易ファイル比較
- `time` — コマンド実行時間計測 (シェル内蔵)

### エディタ

- `edit` — VZ Editorにインスパイアされたテキストエディタ (AIを使用して移植)
  - ギャップバッファ、Undo/Redo、ブロック操作
  - ファイラー、マクロエンジン、ウィンドウ分割

### グラフィックス

- 640×400 16色 CPU直接描画
- libos32gfx ユーザーライブラリ
- サーフェス・スプライト・VBZベクタ形式
- ラスタパレットアニメーション

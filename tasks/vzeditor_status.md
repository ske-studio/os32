# OS32向け VZ Editor 残存課題レポート

## 🔴 実装できていない機能・課題点 (未実装)

1. **マクロエンジンとキーボードカスタマイズ**
   - `macro_engine.c` の実装が開始され、基本的なマクロ実行の枠組みが構築されています。現在は一部のコマンドが静的なマッピングからマクロエンジン経由の呼び出しへ移行中ですが、`DEF`ファイル解釈などの動的なロード機能は未完成です。
2. **大容量ファイル対応の最適化**
   - 現在 `te_resize_buffer` による動的拡張は可能ですが、巨大なファイル（数MB単位）を扱う際の描画速度や、メモリ不足時のスワップ処理などは考慮されていません。

## 今後の方針
エディタとしての基本機能（保存、検索、ブロック操作等）は概ね動作するため、現在は **マクロエンジンの完成** によるカスタマイズ性の向上に注力します。

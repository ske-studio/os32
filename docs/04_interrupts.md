## 第4部 割り込みシステム

### §4-1 IDT構成

| ベクタ | 種別 | 用途 |
|--------|------|------|
| 0x00-0x1F | 例外 | CPU例外 (除算エラー, GPF等) |
| 0x20 | IRQ0 | PIT タイマ (100Hz) |
| 0x21 | IRQ1 | キーボード |
| 0x22-0x27 | IRQ2-7 | マスタPIC (0x27=カスケード) |
| 0x28-0x2F | IRQ8-15 | スレーブPIC |

### §4-2 PIC (8259A) 設定

PC-98のPICはPC/ATと異なるポートアドレスとカスケード構成を持つ。

| 項目 | マスタ | スレーブ |
|------|--------|---------|
| コマンドポート | 0x00 | 0x08 |
| データポート | 0x02 | 0x0A |
| ICW3 (カスケード) | 0x80 (IR7) | 0x07 |
| ICW4 | 0x1D (BUF+SFNM) | 0x09 (BUF) |
| ベクタオフセット | 0x20 | 0x28 |

### §4-3 PIT (8254) 設定

| 項目 | 値 |
|------|-----|
| カウンタ#0 ポート | 0x71 |
| モードレジスタ | 0x77 |
| システムクロック | 1,996,800 Hz (8MHz系) |
| 割り込み周波数 | 100 Hz |
| カウント値 | 19,968 |
| グローバル変数 | `tick_count` (volatile u32) |

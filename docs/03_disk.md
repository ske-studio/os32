### §3-1 フロッピーディスク仕様

| パラメータ | 値 |
|-----------|-----|
| タイプ | 2HD 1MB |
| シリンダ数 | 77 |
| ヘッド数 | 2 |
| セクタ/トラック | 8 |
| セクタサイズ | 1024バイト |
| 総セクタ数 | 1232 (77×2×8) |

### §3-2 ディスクセクタ配置

```
LBA    CHS          用途
─────────────────────────────────────────────
0      C0/H0/S1     IPL (ブートセクタ, 1024B)
1-7    C0/H0/S2-8   stage2 先頭 (loader+pm32+kernel前半)
8-15   C0/H1/S1-8   stage2 続き (kernel)
16-23  C1/H0/S1-8   stage2 続き (kernel)
24-31  C1/H1/S1-8   stage2 続き (kernel)
32-255 C2/H0-C15/H1 kernel 拡張領域 (loader.asmが読込)
256-399 C16-C24      予約 (将来のカーネル拡張用)
400    C25/H0/S1    OS32FS スーパーブロック
401-402 C25/H0/S2-3 OS32FS ディレクトリ (64エントリ)
403    C25/H0/S4    OS32FS ビットマップ
404-1231 C25/H0/S5〜 OS32FS データ領域 (828セクタ)
```

### §3-3 INT 1Bh ディスクI/O

| レジスタ | 読込(06h) | 書込(05h) |
|---------|-----------|-----------|
| AH | 0x76 (SEEK+RETRY+READ) | 0x75 (SEEK+RETRY+WRITE) |
| AL | 0x90 (DA/UA: 1MB FDD unit0) | 0x90 |
| BX | 転送バイト数 | 転送バイト数 |
| CH | セクタ長 (3=1024B) | セクタ長 (3=1024B) |
| CL | シリンダ番号 (0-76) | シリンダ番号 |
| DH | ヘッド番号 (0-1) | ヘッド番号 |
| DL | 開始セクタ (1-8) | 開始セクタ |
| ES:BP | バッファアドレス | バッファアドレス |
| CF | 0=成功, 1=エラー | 0=成功, 1=エラー |
| AH(戻) | ステータスコード | ステータスコード |

### §3-4 HDD (IDE) ディスク仕様

#### NHDイメージのジオメトリ
- **シリンダ数**: 3011
- **ヘッド数**: 8
- **セクタ/トラック**: 17
- **セクタサイズ**: 512バイト
- **総容量**: 約200MB (詳細は [NHD_FORMAT.md](NHD_FORMAT.md) を参照)

#### セクタレイアウト
```
LBA   内容                          サイズ
──────────────────────────────────────────────
  0   IPL (boot_hdd.bin)            512B (1セクタ)
  1   パーティションテーブル         512B (1セクタ)
  2-5 第2ステージローダー            2048B (4セクタ)
  6-  カーネル (kernel.bin)          約80KB (LBA 6〜最大 LBA 271)
──────────────────────────────────────────────
 272  ext2パーティション開始          シリンダ2から
      (スーパーブロック: LBA 274-275)
──────────────────────────────────────────────
```
**注意:** カーネルがパーティション領域（LBA 272以降）に侵食すると `ext2` スーパーブロックを破壊するため、カーネルサイズは最大 133KB に制限されます。

#### パーティションテーブル (LBA 1)
PC-98形式の16バイト/エントリを使用:
- オフセット 0: ブータブルフラグ (`0x80`)
- オフセット 1: システムタイプ (`0xE2` = ext2)
- オフセット 8-9: 開始シリンダ (`2`)
- オフセット 10: 終了セクタ (SPT-1 = `16`)
- オフセット 11: 終了ヘッド (HEADS-1 = `7`)
- オフセット 12-13: 終了シリンダ (CYL-1 = `74`)

### §3-5 INT 1Bh ディスクI/O (HDD仕様)

フロッピーディスクの INT 1Bh とはパラメータの指定方法が**異なります**。

| パラメータ | 指定方法 | 備考 |
|-----------|---------|------|
| DA/UA | `80h` (固定) | SASI/IDE互換アドレス |
| BX | **転送バイト数** | 仕様書には「ブロック数」とあるが、実際には**バイト数を直接指定** (`BX=512`で1セクタ) |
| DL | **セクタ番号** | フロッピーと異なり、HDDでは **0から開始** する (`0〜16`) |
| CX | シリンダ番号 | 全16ビット (CL=下位, CH=上位) |

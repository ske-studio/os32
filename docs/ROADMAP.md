## 4. 長期ロードマップ (v1.0+)

### 16bit DOSプログラム移植スキーム

```
[DOS 16bit .COM/.EXE]
        ↓
  逆アセンブル / 静的解析
        ↓
  INT 21h → KernelAPI マッピング
        ↓
  32bit OS32X として再コンパイル
```

- 半自動 + 手動修正が現実的
- 小さなツール (数KBの .COM) をケーススタディとして開始
- INT 21h → KernelAPI 互換レイヤーの設計が核心

### その他の長期テーマ

- PC-98用NIC (C-bus LAN) → ネットワーク対応
- 協調型マルチタスク
- V30/286機での縮退動作
